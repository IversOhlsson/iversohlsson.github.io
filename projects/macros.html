Trafikverket Exam Slot Notification System

Project Description

During the COVID-19 pandemic, obtaining a driving test slot became challenging due to limited availability. To address this, a macro was designed to scan the Trafikverket website for exam slots. Users could enter a list of potential locations they could travel to for an exam. The macro continuously checked these locations for available exam slots within a given time period and notified users via email when a slot was available. This solution leveraged the flexibility of booking and unbooking up to 24 hours in advance, which meant new slots could become available at any moment.

Smart Mirror with RaspberryPi

Project Description

A Raspberry Pi was programmed to serve as the brain for a smart mirror, which displayed real-time bus schedules from SL.se, as well as other pertinent information like time and weather updates through various APIs. A macro was used to fetch and update the bus arrival times visually, ensuring users were informed about the next bus arrival while going about their morning routine.
