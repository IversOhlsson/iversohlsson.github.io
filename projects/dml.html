Deep Learning Image Coloring Project

Project Description

This project addresses the challenge of adding color to grayscale images by using deep learning models. It employs a U-Net architecture, along with an extended version of VGG for feature extraction, enhanced by the power of transfer learning.

A custom perceptual loss function was implemented to improve the quality and coherence of the generated colored images. The project was developed in Python, utilizing the PyTorch library to build and train the models.

Comparing U-Net and VGG-16 Extended

Abstract

This study evaluates U-Net and extended VGG16 architectures for image colorization on the Imagenette dataset, analyzing performance through mean squared error (MSE) and perceptual loss (PL).

Introduction & Background

We explore image colorizationâ€”a process enhancing grayscale images with color, aiding in fields such as historical media and astronomical data. Imagenette, a subset of ImageNet, is employed due to its manageability. Focus is given to comparing MSE and PL for their effectiveness in colorization.

Methodology

Grayscale images were resized to 224x224 pixels. U-Net was custom-built, while extended VGG16 was enhanced through transfer learning. We assessed the models' performance using quantitative metrics and visual inspections.

Results & Discussion

The results indicate that PL offers a more nuanced colorization compared to MSE. Challenges such as batch size impact and color accuracy were noted, particularly in the U-Net's performance.

Conclusion

Perceptual loss outperformed MSE in producing vibrant and detailed colorizations. The study underlines the need for continued advancements in loss function research and model optimization techniques.

Mean Squared Error (MSE):

$$MSE = \frac{1}{n} \Sigma_{i=1}^{n} (Y_i - \hat{Y}_i)^2$$

Perceptual Loss:

$$\text{Euclidian distance}(x,\hat{x})^2 = (\hat{x} - x)^2$$
