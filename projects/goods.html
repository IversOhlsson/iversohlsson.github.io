Comprehensive Inventory Management System

Project Description

This inventory management system streamlines the process of goods reception and management for businesses. It features a robust front-end built with React, and a secure, scalable back-end powered by Django.

The system leverages MySQL for data storage, ensuring reliable and efficient management of large datasets. The entire application is containerized using Docker, providing consistency across different development and deployment environments, and is hosted on Microsoft Azure for high availability and global scalability.

Users can easily login, register products using a QR scanner that automatically populates relevant product information, and manage their accounts. The system allows for creating and editing user information, offering an intuitive interface for enhanced user experience.
